Voraussetzungen zur Benutzung der Anwendung:

- Java muss installiert sein (über NetInstall)
- die Textdatei "languages.txt" muss im Unterordner "Freischaltungstool-Config"
  (der Ordner wird wenn er nicht vorhanden ist von der Anwendungen erstellt) liegen
- die Initialisierungsdatei "XNIMGRP.CFG" muss im selben Ordner wie die Anwendung liegen
- wenn die Bearbeitungsdauer verändert werden soll (Standard: 15 min), muss in der ersten Zeile der Textdatei "languages.txt" 
  folgender Eintrag erfolgen: 
  close_Time= gefolgt von einer beliebige Bearbeitungsdauer (in Minuten)
