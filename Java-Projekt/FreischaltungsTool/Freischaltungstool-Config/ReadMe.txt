Voraussetzungen zur Verwendung der Anwendung:

- Java muss installiert sein (über NetInstall)
- die Textdatei "languages.txt" muss im Unterordner "Freischaltungstool-Config"
  (der Ordner wird wenn er nicht vorhanden ist von der Anwendungen erstellt) liegen
- die Initialisierungsdatei "XNIMGRP.CFG" muss im selben Ordner wie die Anwendung liegen
- wenn die Bearbeitungsdauer verändert werden soll (Standard: 15 min), muss in der ersten Zeile der Textdatei "languages.txt" 
  folgender Eintrag erfolgen: 
  close_Time= gefolgt von einer beliebige Bearbeitungsdauer (in Minuten)

Requirements to use the application:

- Java has to be installed (over NetInstall)
- the textfile "language.txt" has to lay under the folder "Freischaltungstool-Config"
  (the folder will be created through the application if it doesn't exists) 
- the initialisation-file "XNIMGRP.CFG" must be in the same folder like the application
- if you want to change the remaining work time (default: 15 min), you have to write the following in the first row of 
  the textfile "languages.txt":
  close_Time= followed by your preferred working time (in minutes)
